# 악성 판매자 탐지 (Abusive Seller Detection)

## 🎯 프로젝트 목표

본 프로젝트는 오픈 마켓(Coupang) 내에서 교묘하게 시스템을 악용하는 악성 판매자를 머신러닝을 통해 자동으로 탐지하는 것을 목표로 합니다.
"쿠팡"을 기준으로 데이터를 수집하고 악성 판매자 판별 모델을 학습시켰습니다.

## 🧐 문제 정의 및 접근 방식

쿠팡 등 대형 플랫폼은 이미 가격, 배송 지연, 평점 등의 정량적 지표를 기반으로 한 규칙 기반 필터링 시스템을 갖추고 있을 것으로 추정됩니다.
그러나 최근의 악성 판매자들은 이러한 단순 규칙을 우회하여 정상 판매자처럼 위장하는 패턴을 보입니다.
단순 위장 뿐만 아니라 위너 시스템에서 위너를 차지하는 등, 타사의 트래픽과 매출을 빼앗는 악의적인 행위를 보이기에 이런 악의적인 판매자들을 탐지하는 것을 목표로 합니다.

## 😈 악성 판매자 판별

머신러닝 학습을 위한 정답 데이터(Label)는 휴리스틱(Heuristic) 기반의 정성 평가를 통해 구축했습니다.

- Rule-based의 한계 보완: 가격이나 배송 시간 같은 단순한 수치 기반의 필터링은 플랫폼 차원에서 이미 차단되고 있다고 가정했습니다.
- 인간 지능 기반의 탐지: 따라서 기계적인 필터링을 우회하지만, 실제 소비자가 보기에 직관적으로 의심스러운 판매자(이미지 도용, 리뷰 조작 등)를 식별하는 데 집중했습니다.
- 데이터 일관성: 레이블링의 기준이 흔들리지 않도록 한 명의 검수자가 동일한 기준으로 전체 데이터를 판별했습니다.
