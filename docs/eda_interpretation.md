## 📊 EDA 해석 결과

1. 악성 판매자는 정상 판매자 대비 평균 평점이 낮고, 저평점(1~2점) 비율이 더 높게 나타났다.

2. 부정 키워드(불량, 환불, 가짜 등)를 포함한 리뷰 비율이 악성 판매자에서 유의미하게 높았다.

3. 리뷰 평균 길이는 악성 판매자가 상대적으로 짧은 경향을 보였으며, 유사 리뷰 비율이 높게 나타나 반복적 후기 패턴이 의심된다.

4. 문의 데이터 분석 결과, 악성 판매자는 환불/불량/정품 여부 문의 비율이 정상 판매자보다 높았다.

5. 특히 문의 수 대비 리뷰 비율(question_to_review_ratio)이 비정상적으로 높은 판매자가 악성 판매자로 분류되는 경향이 관찰되었다.

6. 상관관계 분석 결과, 부정 키워드 비율, 저평점 비율, 불량/환불 문의 비율이 타겟 변수와 높은 양의 상관관계를 보였다.

이를 통해 리뷰 및 문의 기반 파생 변수가 판매자 악성 여부 판별에 유효한 특징임을 확인하였다.
